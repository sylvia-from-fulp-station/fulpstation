/tg/ Station 13 - Modules - Types — Var Details - Proc Details

nanite_program
/datum/nanite_program

Vars

activatedBoolean whether the program is currently activated. Off programs won't proceess or have passive effects, but won't consume nanites.
activation_codeCode that activates the program when sent. [1-9999, 0 ignores signals]
all_rules_requiredBoolean on whether all rules are required for positive condition or any of specified.
can_triggerBoolean on whether the nanite program has a trigger function.
deactivation_codeCode that deactivates the program [1-9999, 0 ignores signals]
descThe description of the nanite program for easy understanding.
extra_settingsList of extra settings the nanite program is following.
host_mobThe person hosting this nanite program, if any.
kill_codeCode that permanently removes the program [1-9999, 0 ignores signals]
nameThe name of the Nanite Program.
nanitesThe nanite program component we're connected to.
next_triggerWorld time required for the next trigger activation.
passive_enabledBoolean on whether the nanites have an on/off style effect.
program_flagsSpecial program flags. (NANITE_SHOCK_IMMUNE | NANITE_EMP_IMMUNE)
rogue_typesWhat this program turns into if it glitches. This should be simpler, negative, waste of nanites, or affecting the same bodyparts. Generally.
rulesRules that automatically manage if the program's active without requiring separate sensor programs
timer_restartHow many deciseconds to wait upon deactivation before self-reactivating. Also works if the program begins deactivated.
timer_shutdownHow many deciseconds to wait upon activation before self-deactivating. Also works if the program begins activated.
timer_triggerWhile active, how many dediseconds before the program attempts to trigger itself.
timer_trigger_delayWhile active, how many dediseconds the program will deelay its trigger signals.
trigger_codeCode that triggers the program (if available) [1-9999, 0 ignores signals]
trigger_cooldownDediseconds required between each trigger activation.
trigger_costAmount of nanites this uses when triggered.
uniqueBoolean on whether there can have more than one of this nanite program in the same nanite component.
use_rateHow many nanites this uses while active.
